# 🖥️ Monitor de Temperatura de Servidor

Este proyecto permite monitorear la temperatura de un servidor de forma remota mediante una interfaz cliente-servidor simple. Cuenta con notificaciones por Telegram, para servidor Linux.

---

## 🚀 Características

- 📡 Consulta remota de temperatura del servidor.
- 🔔 Notificaciones automáticas por Telegram si se supera un umbral configurable.


## 📦 Requisitos

- Python 3.12 o superior

---

## ▶️ Uso

Desde el cliente, ejecuta:

```bash
./cliente.py <IP_del_servidor>
```
